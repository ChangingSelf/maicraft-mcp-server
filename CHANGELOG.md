- 新增 `CollectSmeltedItemsAction` 类，支持从熔炉中收集已熔炼完成的物品

- 将原来的 `SmeltItemAction` 删除，使用新的熔炼动作组合

- 更新 README 文档，添加新熔炼动作的使用说明和示例
- 区分了 `CraftItemAction` 对没有配方和材料不足的返回值
